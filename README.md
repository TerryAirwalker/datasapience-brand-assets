# Data Sapience — фирменные ассеты

Исходные файлы для брендбука. Брендбук: https://TerryAirwalker.github.io/datasapience-brandbook/
Машинно-читаемый манифест для ИИ-агента: https://TerryAirwalker.github.io/datasapience-brandbook/manifest.json

## Структура
- `logos/svg/` — логотип DS: полный (`logo-black/white`), знак (`symbol-black/white`), текст (`text-black/white`)
- `logos/png/` — логотип в PNG (2 строки, 3 строки, стек, боковой)
- `logos/products/` — логотипы продуктов (CM Ocean, Kolmogorov, TALYS, Data Ocean Governance)
- `fonts/` — Unbounded (заголовки), Ping LCG (текст)
- `templates/` — `Data Sapience template.pptx` (шаблон презентаций со встроенными шрифтами и макетами)
- `icons/` — линейные иконки: превью (светлая/тёмная) + `icons-png-800.zip` (340 PNG)
- `patterns/` — 12 фирменных паттернов (Blob/Glow 01–03, Blob Filled 04–06, Blob Outline 07–09, Ribbon 10–12)
- `gradients/` — 3 фирменных градиента
- `graphics/` — объёмные 3D-элементы: кольцо, объёмный знак, шестиугольник
