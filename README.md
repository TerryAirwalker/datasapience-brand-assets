# Data Sapience — фирменные ассеты

Исходные файлы для брендбука Data Sapience. Брендбук: https://TerryAirwalker.github.io/datasapience-brandbook/

## Структура
- `logos/svg/` — логотип Data Sapience в SVG: полный (`logo-black/white`), только знак (`symbol-black/white`), только текст (`text-black/white`)
- `logos/png/` — логотип в PNG (2 строки, 3 строки, стек, боковой)
- `logos/products/` — исходники логотипов продуктов (CM Ocean, Kolmogorov, TALYS, Data Ocean Governance)
- `fonts/` — фирменные шрифты: Unbounded (заголовки), Ping LCG (текст)
- `templates/` — `Data Sapience template.pptx` (шаблон презентаций)

## Тексты
Актуальные тексты о компании и продуктах — в мастер-таблице Google Sheets (ссылка в брендбуке, вкладка «Описания»).

Вопросы, доступ, новые ассеты — отдел маркетинга: ms@datasapience.ru (Марике Саар).
